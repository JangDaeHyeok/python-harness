# ADR-0002: 연산적 센서 우선 원칙

## Context

LLM 기반 추론적 센서는 느리고, 비용이 높으며, 비결정적이다.
반면 연산적 센서(린터, 타입 체커, 테스트)는 밀리초~초 단위로 실행되며 결과가 신뢰할 수 있다.

## Decision

Shift Left 원칙에 따라 연산적 센서를 최대한 일찍 실행한다:
1. 린터 (Ruff) → 가장 빠름, 즉시 실행
2. 타입 체커 (mypy) → 초 단위
3. 구조 분석 → 아키텍처 규칙 검증
4. 테스트 (pytest) → 행동 검증
5. AI 코드 리뷰 → 가장 느림, PR 단계에서만 실행

## Consequences

- 비용 효율적: 대부분의 이슈가 저렴한 연산적 센서에서 포착됨
- 빠른 피드백: 에이전트가 즉시 수정 가능
- AI 리뷰는 연산적 센서가 놓치는 의미론적 이슈에만 집중
